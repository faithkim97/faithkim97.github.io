Texture Mapping

Texture Mapping Using UV Coordinates

Tool Used: Three.js ( WebGL JavaScript )

Note: If no 3D model is rendered with relevant texture mapping, then please use FireFox to render the textures, for Google Chrome does not support such properties.

This project consisted of practicing using UV-coordinates to create a checker texture of the octahedron. The checkered image was intentionally rendered to align in a specific through computing the coordinates to UV. The backgrounds were added to see one of my favorite animes be featured in this project.
